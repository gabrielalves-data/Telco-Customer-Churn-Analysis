# Telco Customer Churn Analysis

![CI](https://github.com/gabrielalves-data/Telco-Customer-Churn-Analysis/actions/workflows/test.yml/badge.svg)
![PyPI version](https://img.shields.io/pypi/v/telco_customer_churn_analysis.svg)
[![Documentation Status](https://readthedocs.org/projects/telco_customer_churn_analysis/badge/?version=latest)](https://telco_customer_churn_analysis.readthedocs.io/en/latest/?version=latest)

Developed a financially optimized churn prediction model using a Random Forest classifier and SHAP explainability to identify high-risk customers and maximize retention profit. The solution targets key churn drivers with data-driven interventions, improving customer lifetime value while enabling measurable, ROI-focused campaign deployment.
