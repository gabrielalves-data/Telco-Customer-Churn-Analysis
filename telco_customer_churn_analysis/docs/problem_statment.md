# Problem Statement

A fictional telecom provider with 7,043 customers is losing high-value customers without a predictive mechanism or financial optimization framework.

## Business Challenges
1. **Prediction gap:** No proactive identification of high-risk customers  
2. **Optimization gap:** No method to determine which customers are financially viable to save  

## Goal
Create a system that:
* Predicts churn (maximize Recall)
* Optimizes profit (maximize ROI)
* Provides actionable strategy validated by data
