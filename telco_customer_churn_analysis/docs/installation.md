# Installation

## Stable release

To install Telco Customer Churn Analysis, run this command in your terminal:

```sh
uv add telco_customer_churn_analysis
```

Or if you prefer to use `pip`:

```sh
pip install telco_customer_churn_analysis
```

## From source

The source files for Telco Customer Churn Analysis can be downloaded from the [Github repo](https://github.com/gabrielalves-data/telco_customer_churn_analysis).

You can either clone the public repository:

```sh
git clone git://github.com/gabrielalves-data/telco_customer_churn_analysis
```

Or download the [tarball](https://github.com/gabrielalves-data/telco_customer_churn_analysis/tarball/master):

```sh
curl -OJL https://github.com/gabrielalves-data/telco_customer_churn_analysis/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd telco_customer_churn_analysis
uv pip install .
```
